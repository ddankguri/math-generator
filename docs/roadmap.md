# Roadmap

이 문서는 Math Generator 프로젝트의 공식 개발 로드맵이다.

모든 새로운 기능은 이 로드맵을 기준으로 우선순위를 결정한다.

---

# Project Vision

Math Generator의 목표는

교육과정을 기반으로

무한히 새로운 문제를 생성하는

AI Problem Engine을 구축하는 것이다.

이 프로젝트는

문제를 저장하는 사이트가 아니라

문제 생성 플랫폼을 목표로 한다.

# AI Roadmap

향후

AI는

문제를 생성하는 것이 아니라

유형을 생성한다.

AI 역할

- 공개 자료 분석

- 새로운 유형 제안

- Generator 추천

- Validator 추천

- 난이도 추천

실제 문제 생성은

Engine이 담당한다.

# 성공 기준

Math Generator는

다음을 만족해야 한다.

- 문제를 저장하지 않는다.

- 문제 유형을 저장한다.

- 무한히 새로운 문제를 생성한다.

- 교육과정을 준수한다.

- 정답이 항상 정확하다.

- 풀이를 자동 생성한다.

- PDF를 자동 생성한다.

- AI가 새로운 유형을 추가할 수 있다.

## 개발 우선순위 원칙

새로운 기능을 추가할 때는 아래 우선순위를 따른다.

1. Engine
2. 데이터 구조
3. Validator
4. Solver
5. PDF
6. UI

UI 개선만을 위한 기능 추가는 하지 않는다.

Engine의 품질을 먼저 높인다.
