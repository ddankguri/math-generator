# Future Features

현재 구현 범위 밖의 후보 기능입니다. 목록에 적었다는 이유만으로 사용자 화면이나 sitemap에 미리 노출하지 않습니다.

## 학습 콘텐츠

- 초등 5학년 분수 뺄셈 샘플 단원
- 개념별 선수 학습 연결
- 교사가 활용할 수 있는 지도 질문과 진단 기준 확장
- 실제 화면에 FAQ가 추가될 경우에만 FAQ 구조화 데이터 적용

## 문제 생성

- 생성한 ProblemSet의 브라우저 저장
- 오답 유형에 따른 복습 문제 추천
- Generator·Validator 검증을 거친 추가 학년 지원
- 인쇄 레이아웃과 PDF 접근성 개선

## 운영과 품질

- 실제 브라우저 경로·링크·콘솔 오류 자동 검사
- Open Graph 공유 이미지 제작
- 연락처와 운영 주체 정보를 배포 설정으로 관리
- 광고를 도입할 경우 실질적 본문이 있는 페이지에만 제한하고 오류, 로딩, 빈 결과, 이동 전용 화면에서는 제외

새 기능은 Problem Engine의 Generator → Solver → Validator → Registry 흐름을 변경하지 않고 확장합니다.
